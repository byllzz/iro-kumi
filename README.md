# 🎨 Wada Index
> A digital tribute to the color theory and artistic legacy of Wada Sanzō.

The **Wada Index** is a React-based web archive designed to explore the sophisticated color palettes of Japanese artist Wada Sanzō (1883–1967). This tool translates his 1930s color research into a modern UI for designers and artists.

## 🚀 Features
- **Single Hue Dictionary:** Browse individual traditional Japanese colors.
- **Harmonious Plates:** Interactive 2, 3, and 4-color combinations based on *Haishoku Sōkan*.
- **Click-to-Copy:** Instant hex code extraction for design workflows.
- **Artistic Context:** A gallery connecting palettes to Wada's award-winning costume designs and paintings.

## 🛠️ Tech Stack
- **Frontend:** React.js
- **Routing:** React Router v6
- **Styling:** CSS Modules / Tailwind CSS
- **Data:** JSON-driven architecture

## 📁 Folder Structure
- `src/data`: Contains the JSON color database.
- `src/pages`: Route-level components (Colors, Plates, Gallery).
- `src/components/ui`: Reusable card and button components.
